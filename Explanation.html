Introduction

This application facilitates the marketing department of any company. Usually they have to send newsletters to their customers one by one. Here using this application they can send newsletters with personalized content for each customers by just one click.

Not using newsletters

The newsletter is a great marketing tool. The most important thing for any type of business is to maintain contact with valuable customers, so having a newsletter is a good thing to keep your customers up to date. Customer stays only for a moment on you website therefore it is hard to gain customer's attention before they leave, so attract new customers and entertaining the existing ones is crucial for successful marketing. Approaching the customer with strong pro-active newsletter is a worthy step.

How application works

Step 1: Prepare list of customers, for this you can go to the "Mail Merge" link in website application menu. There you will find a button "Add new Customer", here you will find two fields that are "Name" and "Email", both are mandatory fields. After adding a customer you will see that customer in the list underneath.

Step 2: Once you have entered all of your customers the next thing is to add product and other necessary information shown in figure 1.1. To add product information you need to go to the link "Product Information" in main menu. There you will fill required fields and you are all set to send newsletters.

figure 1.1

Step 3: Finally here you are to send newsletters to all of your customers. Just go to the link "Mail Merger" and click the button "Send this letter to all customers". Suppose you have entered fifty customers in the list, application will create and send 50 Unique documents to customers.

How application was developed

  • Application is developed in Microsoft ASP.NET MVC and C# as back end programming language.
  • SQL Server compact edition is used to manage data in database.
  • Aspose.Words.Reporting for reporting services is used to implement the mail merge and reporting functionality.

Aspose.Words.Reporting

Aspose.Word.Reporting is an API to manipulate Microsoft word documents programmatically. One of the manipulations is to create a word documents with dynamic data provided by user. In current application data is coming from database. Microsoft Word document templates or simple documents are used as templates figure 1.2, in those document(s) we add Merge fields wherever we need dynamic and unique data. Download Template
